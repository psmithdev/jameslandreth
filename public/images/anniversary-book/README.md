# Landreth Family Album Assets

Snapfish does not expose a dependable finished-book PDF export from the builder, so the site is set up to use captured spread images.

Place the final assets here:

- `cover.jpg`
- `spreads/spread-01.jpg`
- `spreads/spread-02.jpg`
- ...
- `spreads/spread-24.jpg`

From Snapfish, use **Spreads** and each spread's **Large view** before capturing. Keep the browser zoom at 100% and make the window as large as possible so the captures are consistent.
